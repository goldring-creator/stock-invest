02

계정 및 API 설정

4개의 API 계정을 순서대로 설정합니다

1
한국투자증권 계좌 + 모의투자 신청

한국투자증권 앱에서 비대면 계좌를 개설합니다. 신분증 촬영만으로 10분 안에 완료됩니다.

계좌 개설 후 트레이딩 → 모의투자 → 모의투자 안내로 이동해 모의투자 계좌를 신청하세요.

한국투자 앱 — 트레이딩 › 모의투자
트레이딩 메뉴에서 모의투자 서브메뉴 선택
한국투자 앱 — 모의투자 안내
참가 신청 버튼을 눌러 모의투자 계좌 개설
2
KIS OpenAPI 앱키 발급

apiportal.koreainvestment.com에 접속해 로그인합니다. 아래 경로로 이동하면 API 신청 화면에 도달합니다.

한국투자증권 — 메뉴 경로
트레이딩 → Open API → KIS Developers 순서로 접근

API 신청 시 모의투자를 선택하고, 해외주식 거래를 원하면 해외주식 포함 옵션을 체크하세요.

KIS Developers — 서비스 신청
모의투자 선택 + 해외주식 포함 체크 후 신청

발급된 앱키(App Key)와 앱시크릿(App Secret)을 메모해두세요.

3
네이버 뉴스 API 신청

developers.naver.com에 접속해 Application을 등록합니다. 사용 API 선택 화면에서 검색을 선택하세요. (뉴스 검색이 포함됩니다)

Naver Developers — API 선택
검색(뉴스 포함) API 선택 후 등록
Naver Developers — Client ID 확인
Client ID와 Client Secret 복사
4
DART 공시 API 인증키 신청

금융감독원의 공시 시스템 opendart.fss.or.kr에서 인증키를 신청합니다. 상장기업의 재무제표를 무료로 수집하는 데 사용됩니다.

DART OpenAPI — 인증키 신청
이름·이메일·사용 목적 입력 후 신청, 이메일로 인증키 수신
5
Telegram 알림 봇 만들기

Telegram에서 @BotFather를 검색하고 /newbot을 입력하면 봇 토큰이 발급됩니다. 봇에게 메시지를 보낸 후 아래 URL에서 chat_id를 확인하세요.

URL
https://api.telegram.org/bot{봇_토큰}/getUpdates

응답 JSON의 result[0].message.chat.id 값이 chat_id입니다.

03

코드 설치 및 설정

GitHub에서 코드를 받아 API 키를 입력합니다

1
저장소 클론 및 의존성 설치
bash
# 코드 다운로드
git clone https://github.com/goldring-creator/stock-invest.git
cd stock-invest

# 의존성 설치
pip install -r 07_코드/requirements.txt
2
설정 파일 작성

06_설정파일/config.yaml 파일을 생성하고 앞서 발급한 키를 모두 입력합니다.

yaml · 06_설정파일/config.yaml
kis:
  mock:
    app_key: "발급받은 앱키"
    app_secret: "발급받은 앱시크릿"
    account_no: "모의투자 계좌번호-01"
  use_mock: true

dart:
  api_key: "DART 인증키"

telegram:
  bot_token: "봇 토큰"
  chat_id: "chat_id"

naver:
  client_id: "Client ID"
  client_secret: "Client Secret"
3
동작 확인
bash
cd 07_코드
python3 scheduler.py --phase=collect
Tip 오류가 발생하면 Python 버전(3.10 이상)과 패키지 설치 여부를 먼저 확인하세요.
04

Claude AI 자동화 연결

AI가 정해진 시각에 자동으로 분석하고 주문을 실행하도록 설정합니다

1
Claude Code 구독

claude.ai/code에 접속해 Claude Code를 구독합니다. Remote Agent 기능을 통해 클라우드에서 AI가 자동 실행됩니다.

2
GitHub App 설치

github.com/apps/claude에서 Claude App을 설치합니다. Remote Agent가 코드 저장소를 읽을 수 있도록 stock-invest 저장소를 선택하세요.

주의 저장소 선택 화면에서 "All repositories" 또는 stock-invest를 명시적으로 지정해야 합니다.
3
자동화 루틴 등록

claude.ai/code/routines에서 루틴 4개를 생성합니다. 각 루틴에 실행 시각과 작업 내용(프롬프트)을 입력하면 AI가 해당 시각에 자동으로 분석을 수행하고 주문을 실행합니다.

프롬프트 파일 위치 저장소 내 07_코드/remote_agent_prompt.md에 각 루틴별 프롬프트가 작성되어 있습니다.
05

AI의 종목 분석 원리

워런 버핏 투자 원칙을 기반으로 AI가 종목을 심사합니다

AI는 감시 종목 목록을 대상으로 5가지 기준으로 점수를 매깁니다. 점수가 높을수록 적극적으로 매수하고, 기준에 미치지 못하면 그 종목은 건너뜁니다.

01 · 경제적 해자
이 기업만이 가진 경쟁 우위가 있는가?
브랜드, 특허, 네트워크 효과 등 경쟁사가 따라오기 어려운 장벽을 평가합니다.
02 · 이해 가능한 사업
10년 후에도 이 사업이 유효한가?
비즈니스 모델이 단순하고 예측 가능한지를 봅니다. 복잡하거나 불확실한 사업은 점수가 낮습니다.
03 · 재무 건전성
수익성과 안정성이 검증되었는가?
ROE, 부채비율, 잉여현금흐름 등 실제 숫자로 기업의 건강함을 확인합니다.
04 · 시장 모멘텀
지금 이 주식이 주목받고 있는가?
외국인·기관 수급, 뉴스 감성, 현재 시장 테마와의 부합도를 분석합니다.
05 · 합리적 가격
기업 가치 대비 현재 주가가 적정한가?
PER, PBR 등 밸류에이션 지표로 현재 주가가 내재가치 대비 어느 수준인지 판단합니다. 아무리 좋은 기업이라도 지나치게 고평가되어 있으면 매수하지 않습니다.

매수 · 매도 판단 기준

매수
  • 5가지 기준 종합 점수가 높은 종목 우선 매수
  • 점수 수준에 따라 매수 수량을 차등 적용
  • 이미 보유 중인 종목은 중복 매수 제외
매도
  • 목표 수익에 도달하면 자동 익절
  • 일정 수준 이상 손실 발생 시 자동 손절
  • 최대 보유 기간 초과 시 자동 청산
06

자동 운영

설정이 완료되면 이 흐름이 매일 자동으로 반복됩니다

오전 09:00
국내 주식 분석 + 매수KR
전날 데이터를 수집하고, AI가 KOSPI/KOSDAQ 감시 종목을 버핏 원칙으로 심사한 뒤 매수 주문을 실행합니다.
오후 03:00
국내 마감 체크 + 일별 리포트KR
장 마감 전 손익 조건을 확인하고 매도가 필요한 종목을 처리합니다. 오늘의 거래 결과를 Telegram으로 전송합니다.
오후 10:00
미국 주식 분석 + 매수US
미국 시장 개장에 맞춰 나스닥·뉴욕 감시 종목을 분석하고 매수 주문을 실행합니다.
새벽 04:30
미국 마감 체크US
미국 장 마감 전 손익 조건을 확인하고 필요 시 매도합니다.
Telegram으로 확인하는 것들 AI가 오늘 어떤 종목을 왜 골랐는지, 매도된 종목과 손익은 얼마인지, 전체 포트폴리오 현황이 매일 자동으로 메시지로 옵니다. 따로 확인하지 않아도 됩니다.
